Функции
    get_user_jwt(request):
        возвращает пользователя по токену
    get_access(action, user):
        получает запрос на действие и возвращает разрешение или запрет на это действие


Практически все api имеют два варианта значения user_id это дефолт и какое то значение(id пользователя),
дефолт предназначен для того чтобы пользователь мог работать со всем функционалом,
а значения используются для получения информации о каком либо пользователе(не только о самом себе)

Для взаимодействия со всеми API нужно указывать token пользователя в header'е Authorization

API:
    cabinet_view(request, user_id='default'):
        Возвращает данные пользователя(если пользователь делал запрос не указывая user_id, то получит информацию
        о себе, иначе, имея соответствующее разрешение получит данные другого пользователя), так же при
        отсутствии профиля поступает ответ, получив который пользователь должен зарегистрироваться

    register_view(request):
        Позволяет создавать профиль для пользователя, а также редактировать информацию о себе

    all_report_view(request, user_id='default'):
        Позволяет посмотреть все отчеты пользователя(свои или не свои), а также создать новый отчет

    report_view(request, report_id, user_id='default'):
        Позволяет посмотреть конкретный отчет пользователя, а также отредактировать его если он принадлежит
        тому, кем был произведен запрос

    all_projects_view(request, user_id='default'):
        Позволяет посмотреть все проекты пользователя, а также создать новый

    project_view(request, project_id, user_id='default'):
        Позволяет посмотреть конкретный проект пользователя, а также отредактировать его

Ответы от api нужно рассматривать отдельно, но как правило если все успешно прошло то возвращается Success
если запрос представляет из себя недопустимый метод, возвращается Method not allowed,
если пользователь пытается что то сделать не имея разрешения, возвращается Permission denied,
также может возвращаться Authentication error если с токеном что то не так(хотя ошибка должна вылететь раньше)
